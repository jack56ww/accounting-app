# 💰 随心记账 - 个人财务管理助手

一个简洁、美观、功能完整的个人记账网页应用，帮助你轻松管理个人财务。

## ✨ 功能特性

### 📊 核心功能
- **收支记录管理** - 添加、编辑、删除收入和支出记录
- **智能分类** - 预设餐饮、交通、购物、工资等16种常用分类
- **数据统计** - 月度收支汇总、分类占比分析
- **可视化图表** - 7天收支趋势图、支出分类饼图
- **预算管理** - 设置月度预算，实时追踪使用进度，超支提醒
- **数据导入导出** - 支持JSON格式数据备份和恢复
- **主题切换** - 支持深色/浅色主题
- **本地存储** - 数据保存在浏览器本地，隐私安全

### 🎨 界面特色
- 现代化简洁UI设计
- 响应式布局，完美适配手机和电脑
- 流畅的动画效果
- 直观的操作体验

## 🚀 快速开始

### 直接使用
直接在浏览器中打开 `accounting-app/index.html` 文件即可使用。

### 本地运行
```bash
# 进入项目目录
cd accounting-app

# 使用任意HTTP服务器运行，例如：
# Python 3
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 然后在浏览器访问 http://localhost:8000
```

## 📝 使用说明

### 添加记录
1. 点击「+ 添加记录」按钮
2. 选择类型（收入/支出）
3. 输入金额、选择分类、选择日期
4. 可选添加备注
5. 点击「保存」

### 设置预算
1. 在预算区域点击「设置预算」
2. 输入月度预算金额
3. 保存后可实时查看预算使用进度

### 数据备份
- 点击右上角「📤」导出数据为JSON文件
- 点击「📥」导入之前备份的数据文件

### 主题切换
- 点击右上角「🌙/☀️」按钮切换深色/浅色主题

## 🔄 Git版本控制

### 提交规范
- `feat:` 新功能
- `fix:` 修复bug
- `docs:` 文档更新
- `style:` 样式调整
- `refactor:` 代码重构
- `chore:` 构建/工具相关

## 🚢 部署到GitHub Pages

1. 在GitHub创建新仓库
2. 关联远程仓库：
```bash
git remote add origin https://github.com/你的用户名/accounting-app.git
git branch -M main
git push -u origin main
```
3. 在仓库设置中启用GitHub Pages
4. 选择main分支作为源，保存后即可在线访问
